Горячие клавиши:

ESC - меню
F1 - этот текст
F2 - сохранить игру (одиночная игра)
F3 - восстановить игру (одиночная игра) / дипломатия (сетевая игра)
<~~> или <I> - открыть/закрыть рюкзак
<Q> или <B> - открыть/закрыть книгу заклинаний
<W> - открыть/закрыть список заданий (сетевая игра)
Пробел - открыть/закрыть рюкзак и книгу заклинаний / оживить персонажа после смерти (сетевая игра)
TAB - параметры/картинка персонажа
Ctrl+<O> - сглаживание вкл/выкл
Ctrl+<H> - отображать жизнь для всех вкл/выкл
Ctrl+<L> - отлетающий урон вкл/выкл
Ctrl+<W> - изменить режим отступления
Ctrl+<F> - изменить режим сохранения строя
Ctrl+<N> - смена дня и ночи вкл/выкл
Ctrl+<U> - изменить режим автоматического применения заклинаний
Ctrl+<K> - отображать имена и кланы вкл/выкл (сетевая игра)
Ctrl+<T> - измерение траффика вкл/выкл (сетевая игра)
Ctrl+<C> - цвета сообщений стандартные/альтернативные
Серые <+>,<-> - изменить скорость игры
<M> - идти
<A> - атаковать
<G> - охранять
<D> - защищать
<C> - колдовать
<S> - идти в боевой готовности
<T> - держать позицию
<R> - отступать
<P> - собирать мешки (только для героев)
<L> - стоять смирно (не атаковать врагов)
<E> - выбрать всех ваших персонажей
ALT - форсировать команду идти
CTRL - форсировать команду атаковать/идти в боевой готовности
SHIFT - форсировать добавление в группу/выделение из группы
CTRL+<1..9> - запомнить выбранную группу (одногрупповой режим)
CTRL+SHIFT+<1..9> - запомнить выбранную группу (многогрупповой режим)
<1..9> - выбрать запомненную группу
ALT+<1..9> - отцентрировать экран на запомненную группу
SHIFT+<1..9> - добавить группу к выбранной группе
Ctrl+F4..F12 - установить горячую клавишу для заклинания
Shift+F4..F12 - установить горячую клавишу для предмета
F4..F12 - колдовать/использовать предмет по горячей клавише
Ctrl+A - назначить/снять боевое заклинание для автоматического использования
ENTER - послать сообщение (в режиме сетевой игры)
BACKSPACE - стереть все сообщения
ALT+F4 - аварийный выход из игры

Режим набора сообщений (в режиме сетевой игры):

BACKSPACE - удалить символ слева
Ctrl+BACKSPACE - удалить слово слева
TAB - подставить имя игрока по нескольким буквам слева от курсора
<Стрелка Вверх> - вспомнить предыдущее сообщение
[message] - сообщение для всех, кто находится рядом
=[message] - сообщение для всех играющих
-[message] - сообщение для всех, кто находится с вами в альянсе
-<имя игрока>[message] - сообщение конкретному игроку
#kick <имя игрока> - удалить игрока из игры
#locate <имя игрока> - найти местоположение игрока
